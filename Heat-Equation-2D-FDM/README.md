# 2D Heat Equation using Explicit Finite Difference Method

## Description

This project solves the two-dimensional heat equation using the Explicit Finite Difference Method (FTCS).

## Governing Equation

du/dt = α(d²u/dx² + d²u/dy²)

## Initial Condition

A hot square is placed at the center of the computational domain.

## Boundary Conditions

- Temperature is zero on all boundaries.

## Features

- MATLAB implementation
- Explicit FTCS scheme
- 2D heat diffusion
- Surface plot visualization
- Colorbar
