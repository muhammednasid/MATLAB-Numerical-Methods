# 2D Laplace Equation using Finite Difference Method

## Description

This project solves the two-dimensional Laplace equation using the Finite Difference Method (FDM) and the Jacobi iterative method.

## Governing Equation

∂²u/∂x² + ∂²u/∂y² = 0

## Boundary Conditions

- Left Boundary = 100
- Right Boundary = 50
- Top Boundary = 75
- Bottom Boundary = 25

## Numerical Method

- Finite Difference Method
- Jacobi Iterative Method
- Uniform Grid

## Features

- MATLAB implementation
- Surface plot visualization
- Filled contour plot
- Convergence criterion

## Output

- Surface plot
- Contour plot
